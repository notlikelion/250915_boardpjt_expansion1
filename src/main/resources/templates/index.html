Spring Boot 게시판 프로젝트

이 프로젝트는 Spring Boot를 기반으로 현대적인 웹 애플리케이션의 핵심 기능들을 구현한 게시판입니다. 사용자 인증부터 게시물 관리, 소셜 기능까지 다양한 기술을 체험할 수 있습니다.

게시글 보러가기

주요 기술 스택

  • 백엔드: Spring Boot, Spring Security, Spring Data JPA
  • 데이터베이스: MySQL, Redis (토큰 관리)
  • 인증: JWT (Access/Refresh Token)
  • 프론트엔드: Thymeleaf, Bootstrap
  • 빌드: Gradle

구현된 기능

  • 회원가입 및 로그인 (JWT 기반 인증)
  • 게시글 CRUD (생성, 조회, 수정, 삭제)
  • 댓글 작성 및 삭제 (AJAX)
  • 페이지네이션 및 검색 기능
  • 사용자 팔로우/언팔로우
  • 관리자 페이지 (회원 관리)
  • 비밀번호 변경 및 회원 탈퇴
